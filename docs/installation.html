Installation


Navi supports linux platform only.

For now, Navi can only be built from source. The following dependencies have to be installed first for Navi to work properly:
  • Qt6
  • Lua
  • ImageMagick (Magick++)
  • Poppler (Poppler-Qt6)
  • Udisks

If you're on ArchLinux you can copy paste the following pacman code to install these dependencies:
sudo pacman -S qt6-base lua imagemagick poppler poppler-qt6 udisks2
If you're on Ubuntu/Debian based distribution, you can paste the following apt code to install the dependencies:
sudo apt install qt6-base-dev lua5.4 libmagick++-dev libpoppler-dev libpoppler-cpp-dev libpoppler-qt6-dev udisks2 libtree-sitter-dev
NOTE: I have not tested Navi on an Ubuntu machine. If there are any problems, please open an issue.

After installing these dependencies on your system and cloning this project repository, you can build navi by using the following command (inside the projectâ€™s directory):
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make -j$(nproc)
sudo make install
One Liner:
mkdir build && cd build && cmake -DCMAKE_BUILD_TYPE=Release .. && make -j $(nproc) && sudo make install
If there were no errors, you will be able to run navi either from the commandline or a run launcher by simply calling navi

Check out docs for help on using navi and it's many configuration settings.
